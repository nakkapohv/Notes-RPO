#### Введение

**WinGet** - это официальный менеджер пакетов от Microsoft для Windows 10, Windows 11 и Windows Server 2025. Это инструмент командной строки (CLI), который позволяет пользователям автоматизировать процесс поиска, установки, обновления и удаления приложений, централизованно управляя программным обеспечением на компьютере.
#### Установка и доступность 

WinGet предустановлен в Windows 11 и современных версиях Windows 10 в составе компонента «Установщик приложений» (App Installer) из Microsoft Store. Для запуска используется команда `winget` в терминале Windows, PowerShell или командной строке,

#### Базовые команды

Утилита предлагает интуитивные команды для управления софтом. Например:
- winget search <запрос> для поиска
- winget list для просмотра установленных программ
- winget install ID для установки
- winget upgrade --all для обновления всех приложений 
- winget uninstall ID для удаления 
#### Поддерживаемые форматы 

WinGet работает с большинством распространенных типов установщиков, включая MSI, EXE, MSIX, APPX и установщики Inno Setup, Nullsoft и WiX.

#### Пакетная установка и конфигурация

Для автоматизации настройки новой системы WinGet использует файлы конфигурации в формате YAML. Команда winget configure позволяет описать желаемое состояние системы (набор программ и параметров), после чего менеджер пакетов автоматически приведет компьютер в заданное состояние

#### Заключение

WinGet - это мощный инструмент, который превращает установку и обслуживание софта в Windows в быстрый и автоматизированный процесс. Он незаменим для разработчиков, ИТ-администраторов и продвинутых пользователей, позволяя экономить время, обеспечивать идентичность окружения на разных машинах и легко обновлять все программы одной командой
