#### Введение 

Обычно установка Windows 10 происходит с флешки, если раньше надо было заморачиваться, то со временем этот процесс стал почти автоматическим и зная пошаговую инструкцию - любой сможет справиться.

##### Пошаговая инструкция:

- Для начала качаем с официального сайта образ ISO с нашей windows
- Далее подготавливаем флешку, а точнее:
  1) Качаем программку rufus, с помощью которой мы форматируем нашу флешку под установку нашего windows
- Сохраняем все нужные файлы, освобождаем один из дисков полностью или если один то делим его на два 
- !!!! Обязательно нужно запомнить на каком диске лежит информация и запомнить память дисков !!!!!!!
- Далее после того как все нужные файлы в безопасности, перезагружаем ПК
- Во время перезагрузки спамим кнопку которая откроет наш BIOS (f2, del, f10), далее выбираем запуск с нашего носителя (а точнее с флешки где наша Winodws)
- После этого ждём, весь процесс будет автоматическим 
- После чего нас спросят какую Windows мы хотим (PRO, Home и т.д), выбираем нужную нам
- После чего нас запросят форматировать один из дисков (а точнее мы выбираем диск, который мы выделили для установки Windows и нажимаем снизу форматировать и тыкаем далее)
- Дальше нам предложат назвать свой профиль Windows (который является админом по умолчанию), поставить пароль для безопасности, авторизоваться в Microsoft и ввести ключ доступа
- Готово!
  
  
#### Основные особенности установки Windows 10 на виртуальную машину

1. Выбор гипервизора (программы для виртуализации)
Наиболее популярные:

- Oracle VirtualBox 
- VMware Workstation Pro 
- Hyper-V 

2. Требования к системе
Чтобы виртуальная машина работала плавно, нужно выделить достаточно ресурсов:

Компонент	                                     Минимум	          Рекомендуется

Оперативная память (RAM)	              2 ГБ	                    4–8 ГБ

Процессор (CPU)	                            1 ядро	                2 ядра и более

Жёсткий диск (HDD/SSD)	                32 ГБ	                 60–100 ГБ

Видеопамять (VRAM)	                        128 МБ	            128–256 МБ (с 3D-ускорением)

3. Подготовка образа Windows 10 (ISO-файл)

Нужен официальный ISO-образ Windows 10. Его можно скачать с сайта Microsoft. Выберите 64-битную версию, если ваш ПК поддерживает (рекомендуется).

4. Настройка виртуальной машины перед установкой
После создания виртуальной машины — обязательно настройте параметры:

- Выделить 2 процессора (ядра).
- Включить PAE/NX (для лучшей совместимости).
- Увеличить видеопамять до 128 МБ.
- Включить 3D-ускорение, если планируем использовать графику.
- Активировать VT-x/AMD-V (аппаратная виртуализация) это значительно ускорит работу

5. Процесс установки — пошагово

- Запустите виртуальную машину.
- В настройках указать ISO-образ Windows 10 (через раздел “Хранилище” → “Оптический привод”).
- Машина загрузится с образа.
- Выберите язык, формат даты и клавиатуры.
- Нажмите “Установить”.
- Примите лицензионное соглашение.
- Выберите “Выборочная: только установка Windows”.
- Установка начнётся — машина перезагрузится несколько раз.
