#Утилита #NetUser #УтилитаNetUser #ОсновныеКоманды #Определение 
#### Введение

Утилита net user — это мощный инструмент командной строки Windows для управления локальными учетными записями: просмотра, создания, удаления и редактирования пользователей. Для использования запустите командную строку (cmd) или PowerShell от имени администратора. 

#### Основные команды

1) Просмотр списка пользователей - net user (команда), вставляем в CMD
2) Изменение пароля пользователя - net user (имя) (пароль), вставляем в CMD
3) Просто данных конкретного пользователя  - net user (имя_пользователя), вставляем в CMD
4) Просмотр локальных групп - net localgroup, вставляем в CMD или же net localgroup (имя_группы): показывает список пользователей, входящих в конкретную локальную группу
5) Назначение пользователя в локальную группу - net localgroup (имя_группы) (имя_пользователя) /add
6) Запрет на изменение пароля - net user (имя_пользователя) / passwordchg:no
7) Удаление пользователя -  net user (имя_пользователя) /delete
