## Определения

**`.bat` (batch file)** — текстовый скрипт командной строки для `cmd.exe`, выполняющий последовательность команд. Используется для автоматизации задач, установки окружения, запуска программ и простого администрирования. #bat

Ключевые определения
- **Переменная** — именованная ячейка, хранящая строку; обращение: `%NAME%`.
- **Код выхода (exit code)** — целое число, которое процесс возвращает при завершении; 0 = успех, не 0 = ошибка. (`%ERRORLEVEL%` в `cmd`).

## Советы

1. Используйте метки (`:label`) и `call` для разбивки на логические блоки.
2. **Защита контекста**: `setlocal` / `endlocal` — чтобы изменения переменных не «утекали» наружу.
3. Всегда оборачивайте пути в кавычки: `"%~dp0"` — путь к каталогу скрипта.
4. После важных команд проверяйте `%ERRORLEVEL%` или используйте: `&&` / `||`.
     - Пример: `command || (echo Ошибка & exit /b 1)`
5. **Проверка наличия утилит**: `where program.exe` — чтобы убедиться, что исполняемый файл в `PATH`.
6. **Комментарии**: `rem` или `::` для пояснений.
