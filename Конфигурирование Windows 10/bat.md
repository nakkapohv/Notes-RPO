#bat #cmd #батфайл #бат 
### Введение

.bat (бат-файл) — это исполняемый текстовый документ для командной строки Windows (cmd.exe). Он содержит последовательность команд, которые операционная система выполняет построчно. Основное назначение скрипта это автоматизация рутинных процессов: пакетная обработка файлов, настройка рабочего окружения и последовательный запуск приложений.
### Принцип работы 

1) В обычном текстовом редакторе (например, в «Блокноте») записывается список нужных действий — команд (например, запустить программу, скопировать папку или очистить кэш).
2) Текстовый файл сохраняется, но вместо стандартного расширения `.txt` на конце указывается `.bat`.
3) Файл запускается двойным кликом левой кнопки мыши, как обычная программа.
4) В этот момент открывается черное окно командной строки Windows. Система читает первую строчку текста в файле и выполняет её. Затем читает вторую и выполняет её. И так далее, строго сверху вниз.
5) Процесс полностью аналогичен ручному вводу каждой команды по очереди с нажатием Enter. Бат-файл просто автоматизирует эту работу и делает её за доли секунды. Как только система выполняет последнюю строчку, черное окно (cmd) автоматически закрывается.
