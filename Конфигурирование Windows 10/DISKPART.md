#Diskpart #РаботаСDiskpart

Я работал с DISKPART с помощью нейросети, он выдал мне задачи, которые я выполнял.

1) Инициализируйте диск как GPT
- List disk
- select disk 0
- Clean
- Convert GPT
2) Создайте основной раздел размером 100 ГБ
- Select disk 0
- create partition primary size=100
3) Отформатируйте его в NTFS, назначьте метку «Data»
- List volume
- Select volume 0
- Format fs=ntfs label=Data
1) Назначьте разделу букву F
- Select volume 0 
- assign letter=F

#### Основные команды DISKPART

ACTIVE       - Пометить выбранный раздел как активный.
ADD          - Добавить зеркало к простому тому.
ASSIGN       - Назначить букву диска или точку подключения выбранному тому.
ATTRIBUTES   - Работа с атрибутами тома или диска.
AUTOMOUNT    - Включение/отключение автоматического подключения базовых томов.
BREAK        - Разбить зеркальный набор.
CLEAN        - Удалить все разделы или том с диска.
COMPACT      - Попытка уменьшить физический размер файла.
CONVERT      - Преобразование форматов дисков.
CREATE       - Создание тома, раздела или виртуального диска.
DELETE       - Удаление объекта.
DETAIL       - Вывод сведений об объекте.
DETACH       - Отсоединение файла виртуального диска.
EXIT         - Завершение работы DiskPart.
EXTEND       - Расширение тома.
EXPAND       - Расширение виртуального диска до указанного размера.
FILESYSTEMS  - Отображение текущей и поддерживаемых файловых систем для тома.
FORMAT       - Форматирование тома или раздела.
GPT          - Назначение атрибутов выбранному GPT-разделу.
HELP         - Вывод списка команд.
IMPORT       - Импорт группы дисков.
INACTIVE     - Пометить выбранный раздел как неактивный.
LIST         - Вывод списка объектов.
MERGE        - Объединение дочернего диска с родительским.
ONLINE       - Перевод объекта в состояние "в сети".
OFFLINE      - Перевод объекта в состояние "не в сети".
RECOVER      - Обновление состояния всех дисков выбранного пакета.
REM          - Однострочный комментарий.
REMOVE       - Удаление буквы диска или точки подключения.
REPAIR       - Восстановление тома RAID-5.
RESCAN       - Повторное сканирование дисков и томов.
RETAIN       - Размещение системного раздела на простом томе.
SAN          - Отображение или установка политики SAN для текущей загруженной ОС.
SELECT       - Перемещение фокуса на объект.
SETID        - Изменение типа раздела.
SHRINK       - Уменьшение размера выбранного тома.
UNIQUEID     - Отображение или установка идентификатора GPT-раздела или сигнатуры MBR-диска.
