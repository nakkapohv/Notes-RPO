#SSH 
### Введение

SSH (Secure Shell) — это сетевой протокол транспортного уровня, который позволяет производить безопасное удалённое управление операционной системой и туннелирование TCP-соединений. Все данные, передаваемые по протоколу (включая пароли), шифруются, что защищает их от перехвата.
### Применение

Протокол SSH используется для решения широкого спектра задач в администрировании и разработке:

- Удалённое управление: Доступ к терминалу серверов (Linux/машины на macOS) с домашнего компьютера.
- Безопасная передача файлов: Использование протоколов **SFTP** (Secure File Transfer Protocol) и **SCP** (Secure Copy) для копирования файлов между устройствами.
- Туннелирование (Port Forwarding): Проброс портов для безопасного доступа к локальным сервисам удалённого сервера (например, к базе данных, закрытой брандмауэром).
- Управление Git-репозиториями: Аутентификация на GitHub, GitLab и других сервисах без постоянного ввода логина и пароля (по SSH-ключам).
