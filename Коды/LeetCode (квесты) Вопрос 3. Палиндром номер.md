Для целого числа , _возвращается, если_ _—_ `x``true``x`_**Палиндром**__, и_ _не только_.`false`


```` Python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        p = str(x)
        a = []
        for i in p:
            a += i
        if a == a[::-1]:
            return True
        else:
            return False
````

Создаём класс, в котором создаём функцию, внутри которой:

1) Преобразуем переменную x, которая изначально задана как целое число int, в строку в переменную p
2) Задаём в переменную a - пустой список
3) Далее перебираем для переменной i в p: в список присваиваем i 
4) Далее, если список a (обновленный), равен перевёрнутому списку, возвращаем True
5) Иначе, возвращаем False
