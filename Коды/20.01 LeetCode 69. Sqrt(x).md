
Для неотрицательного целого числа вернём квадратный корень из округлённого вниз до ближайшего целого числа. Возвращаемое целое число также должно быть неотрицательным.xx

Нельзя использовать встроенную функцию или оператор экспоненты.


```` Python
class Solution:
    def mySqrt(self, x: int) -> int:

        return int(x ** 0.5)
````

Самое просто решение этой задачи, мы возводим x в степень 0.5, это тоже самое что изъять корень из числа и возвращаем это всё целым числом (округленным).
