```` Python
class Solution:
    def pivotInteger(self, n: int) -> int:

        #Формула Гаусса для нахождения общей суммы
        res0 = (n * (n+1) // 2)

        #Находим квадратный корень от общей суммы
        res1 = int(res0**0.5)

        #Если произведение корней равно общей сумме, вернуть корень
        if res1 * res1 == res0:
            return res1

        #Если условие не сработает, вернётся -1
        return -1 
 
````

Сначала создаём класс, в котором задаём функцию, внутри которой:

1) Присваиваем переменной res0 - формулу Гаусса для нахождения общей суммы
2) Находим квадратный корень от общей суммы, с помощью присваивания переменной res1 - математического действия в котором переменную res0 мы возводим в степень 0.5
3) Далее делаем проверку, если произведение корней равно общей сумме, то мы возвращаем корень
4) Если наша проверка False, то идём дальше и возвращаем -1
