#Код
Имея строку, состоящую из слов и пробелов, возвращаем длину последнего слова в строке.s

Слово — это максимум Подстрока состоящий только из символов, не размещающихся в пробелах.


```` Python
class Solution:
    def lengthOfLastWord(self, s: str) -> int:
         words = s.split()
         return len(words[-1])
````

Достаточно простой код

Создаём класс, в котором задаём метод/функцию в который принимает строку s, а возвращает целое число

Далее задаём список words, в котором строку разбиваем строку на список слов, т.к мы не указываем никаких аргументов .split() - удаляет лишние пробелы 

Далее возвращаем длину последнего слова.
