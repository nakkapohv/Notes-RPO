Пример кода на Python:

```` Python 
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        i = len(a) - 1
        j = len(b) - 1
        carry = 0 
        result = []
        while i >= 0 or j >= 0 or carry:
            x = int(a[i]) if i >= 0 else 0 
            y = int(b[j]) if j >= 0 else 0 

            total = x + y + carry
            result.append(str(total % 2))
            carry = total // 2

            i -= 1
            j -= 1
        return ''.join(result[::-1])
````

Объяснение:

Создаём класс, в котором формируем функцию.

Сначала задаём переменные:

i - в которой мы строку a читаем с конца 
j -  в которой мы строку b читаем с конца
carry - которая будет переносить из предыдущего разряда 0 или 1 
result - создаём пустой список

Задаём цикл While:

Когда i больше или равно 0 или j больше или равно 0 или carry:
то, если индекс i или j вышел за границы строки, берём 0 (если одна строка короче другой)
Преобразуем строки '0' или '1' в числа 

Задаём total, где складываем все переменные 
Дальше в результат добавляем, но только если чётное
Задаём, переменную carry, только в том случае если сумма делится на цело 

Перебираем значения строк a и b, если i или j < 0, то значит что строка кончилась.

Далее с помощью return, объединяем все символы в строку и переворачиваем обратно, т.к до этого мы рассматривали в обратом порядке)
