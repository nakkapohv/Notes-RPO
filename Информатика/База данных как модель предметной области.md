#### Определение

База данных (БД) - это совокупность данных организованных по определенным правилам, отражающее состояние объектов и их отношений, в некоторой предметной области. Предназначенная для хранения во внешней (постоянной) памяти и её постоянного применения. Предметная область - целевая тема конкретного проекта. В свою очередь доменная или концептуальная модель призвана выразить значение терминов и понятий, а также установление верных взаимосвязей между различными понятиями. Базу данных можно рассматривать как информационную модель предметной области. 
#### Три типа базы данных

- Реляционные базы данных (табличные) - это тип баз данных, которые хранят данные в виде таблиц. Каждая таблица состоит из строк (записей) и столбцов. Эти таблицы связаны друг с другом.
- Сетевые базы данных - в сетевых базах данных между таблицами может быть несколько различных связей, в том числе несколько предков. Особенной этой базы в том что в ней запоминаются все связи и всё содержимое для каждой из них. Иначе говоря, поиск в такой базе данных индексируется. 
- Иерархические базы данных - это модель организующие данные в древовидную структуру, где каждый дочерний элемент подчиняется только одному родительскому.

Таким образом база данных это место (база), где хранятся все наши данные. Они бывают разных видов: реляционные, т.е разные таблицы связаны между собой при помощи ID (универсального номера), сетевые - в них запоминаются все связи и все содержимое и иерархические, которые работают по принципу древа, т.е от каждой главной идут дочерние, которые подчиняются только одному родительскому.
