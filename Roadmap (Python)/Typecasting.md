### Введение

Преобразование типов — это процесс изменения типа данных объекта на другой (например, превращение числа в текстовую строку или наоборот). В Python этот механизм бывает двух видов: неявный (работает автоматически) и явный (контролируется вручную).
### Неявное преобразование

Этот вид преобразования выполняется интерпретатором Python самостоятельно, без участия программиста. Главное правило автоматики — защита информации от искажений: чтобы избежать потери данных, Python всегда повышает «меньший» тип данных до «большего». Например, если вы сложите обычное целое число (`int`) и число с плавающей точкой (`float`), итоговый результат автоматически примет формат `float`. Однако автоматика работает не всегда: программа выдаст ошибку типа (`TypeError`), если попытаться напрямую сложить текст и число.
### Явное преобразование

Это ручное изменение типа данных, которое разработчик задает принудительно с помощью встроенных функций, таких как `int()`, `float()` или `str()`. Если нужно решить проблему из предыдущего пункта и сложить строку `'12'` с числом `23`, программист должен сначала пропустить строку через функцию `int()`. После такого преобразования Python увидит два полноценных числа и успешно их сложит. Процесс явного преобразования также называют «кастованием». Важно учитывать, что при ручном приведении типов может произойти потеря данных, так как мы силой заставляем объект принять новый формат (например, перевод дробного числа во `float` в целое `int` просто отрежет его дробную часть).
