#Lists #Списки #Pandas

### Введение

Списки — это фундаментальная структура данных в Python, используемая для хранения упорядоченного набора элементов. Эти элементы могут быть разных типов (числа, строки или даже другие списки). Списки изменяемы, то есть вы можете менять их содержимое после создания, добавляя, удаляя или изменяя элементы. Списки обозначаются квадратными скобками `[]`, а элементы разделяются запятыми.
### Основы списков

В отличие от обычной переменной, которая хранит только одно значение, список позволяет хранить множество элементов под одним именем. Для его создания элементы перечисляются через запятую внутри квадратных скобок, например: `color = ['blue', 'green', 'red', 'yellow']`. Важно помнить, что в программировании отсчет позиций всегда начинается с нуля, а не с единицы. Таким образом, элемент `blue` имеет индекс 0, а `green` — индекс 1. Чтобы получить доступ к конкретному элементу, его индекс указывается в квадратных скобках рядом с именем списка, и команда `print(color[1])` выведет на экран значение `green`.

### Взаимодействие нескольких списков

Списки можно удобно комбинировать друг с другом, связывая их данные. Если в программе создать второй список с названиями фруктов, то можно легко выводить пары значений, просто обращаясь к одинаковым позициям в обоих массивах. Например, объединив элементы под первым индексом с помощью команды `print(color[1], fruit[1])`, вы получите логичный результат `green apple`.

### Организация данных с помощью Pandas

Если возникает необходимость вывести несколько списков не сплошной строкой, а в виде красивой и читаемой таблицы со столбцами, базового функционала языка недостаточно. Для таких задач используется сторонний фреймворк для анализа данных под названием Pandas. Библиотека требует предварительной установки, после чего импортируется в код командой `import pandas as pd`. Главным инструментом здесь выступает `DataFrame` — специальная функция для работы с двумерными табличными массивами. Процесс создания таблицы начинается с формирования структуры столбцов конструкцией `df = pd.DataFrame(columns=['color', 'fruit'])`, после чего этим столбцам присваиваются значения из ранее созданных списков. При выводе такой переменной на экран с помощью `print(df)` данные отобразятся в виде аккуратной таблицы с автоматической нумерацией строк.

```` Python
#Import the Pandas framework, defined as pd
import pandas as pd

#Define our color variable as a list
color = ['blue','green','red','yellow']
#Define our fruit variable as a list
fruit = ['blueberry','apple','cherry','banana']

#Define the df variable as formatted columns for color and fruit
df=pd.DataFrame(columns=['color', 'fruit'])
#Label our columns as color and fruit
df['color'],df['fruit']=color,fruit

#Print everything
print(df)

#Выход
0   color    fruit
1   blue     blueberry
2   green    apple
3   red      cherry
3   yellow   banana
````
