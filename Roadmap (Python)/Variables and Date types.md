### Введение

Переменные используются для хранения информации, на которую можно ссылаться и которой можно управлять в компьютерной программе. Они также позволяют присваивать данным описательные имена, чтобы читателю и нам самим было проще понимать наши программы. Переменные можно представить как контейнеры для хранения информации. Их единственная цель — присваивать имена данным и хранить их в памяти. Эти данные можно использовать в любой части программы.
### Создание и динамическая типизация

Переменные в Python создаются автоматически в момент первого присваивания им значения с помощью знака равенства `=`. Вам не нужно заранее объявлять переменную или указывать, какие данные в ней будут лежать. Язык использует **динамическую типизацию**: тип переменной определяется сам на лету по типу присвоенного значения. Тип можно менять прямо по ходу программы: если сначала в `data` лежало число (`data = 10`), то следующей строкой туда можно без проблем записать текст (`data = "Тест"`).
### Правила и стиль именования переменных

Имя переменной может состоять из латинских букв, цифр и символа подчеркивания `_`. При этом имя обязательно должно начинаться с буквы или с `_` (начинать с цифры запрещено). В Python строго учитывается регистр: `user`, `User` и `USER` — это три абсолютно разные переменные. Также нельзя называть переменные системными ключевыми словами (например, `if`, `else` или `while`). По общепринятому стандарту кода PEP 8 переменные принято называть строчными буквами в стиле **snake_case** (слова разделяются нижним подчеркиванием), например: `user_age`, `total_sum`.
### Множественное присваивание

Python позволяет очень компактно работать с переменными. Вы можете присвоить одно и то же значение сразу нескольким переменным в одну строку: `a = b = c = 0`. Также можно объявить несколько разных переменных одновременно через запятую: `name, age = "Иван", 25`. Благодаря этому свойству в Python можно поменять значения двух переменных местами в одну строчку без использования стандартной третьей буферной переменной: `a, b = b, a`.

```` Python
favorite_food = "Пицца"  # Сначала храним текст
favorite_food = 100      # Теперь храним число, ошибки не будет
````

```` Python
# ПРАВИЛЬНО:
tasty_burger = "Чизбургер"
sushi_roll_2 = "Филадельфия"

# НЕПРАВИЛЬНО (начинается с цифры, выдаст ошибку):
# 1_slice = "Пицца"
````

```` Python
Food = "Бургер"
food = "Суши"

# Это две абсолютно разные переменные
````
