### Введение

Строки в Python — это последовательности символов, используемые для представления текста. Их можно создать, заключив символы в одинарные, двойные или тройные кавычки. После создания строки становятся неизменяемыми, то есть их значения нельзя изменить напрямую. К распространенным операциям относятся доступ к отдельным символам с помощью индексации, извлечение подстрок с помощью срезов, объединение строк с помощью оператора `+`, а также использование встроенных методов для обработки и форматирования содержащегося в них текста.
### Срезы и тип работы

Поскольку строка является последовательностью, к каждому ее символу можно обратиться по порядковому номеру, который называется индексом. Прямая индексация начинается строго с нуля, а обратная позволяет читать с конца, где минус один обозначает последний символ. Для извлечения кусков текста используются срезы с синтаксисом `[старт : стоп : шаг]`, при этом правая граница никогда не включается в итоговый результат. Например, конструкция `[::-1]` элегантно развернет строку задом наперед.
### Базовые операции и форматирование

Строки поддерживают некоторые математические операторы: их можно склеивать друг с другом с помощью знака плюса (конкатенация) или дублировать, умножая на целое число. Для вставки значений переменных прямо в текст самым современным и читаемым способом являются f-строки. Для их создания перед кавычками ставится английская буква `f`, а нужные переменные или выражения оборачиваются в фигурные скобки прямо внутри текста.
### Инструменты

Cуществуют десятки встроенных методов для обработки текста. Системная функция `len()` возвращает общую длину строки. Методы `.lower()` и `.upper()` переводят все буквы в нижний или верхний регистр. Метод `.strip()` очищает текст от невидимого мусора вроде лишних пробелов по краям, а `.replace()` находит и заменяет нужные участки. Для изменения структуры данных применяются `.split()`, разрезающий одну строку на список элементов по заданному разделителю, и `.join()`, который выполняет обратное действие, собирая список отдельных строк в один цельный текст.

```` Python
text = "Привет"

print(text[0])     # Результат: П (первая буква)
print(text[-1])    # Результат: т (последняя буква)
print(text[0:4])   # Результат: Прив (срез, правая граница не берется)
print(text[::-1])  # Результат: тевирП (строка задом наперед)
````

```` Python
# Конкатенация (сложение) и умножение
print("Ха" * 3)  # Результат: ХаХаХа

# f-строки (современная вставка)
name = "Иван"
age = 20
print(f"Меня зовут {name}, мне {age} лет.")
````

```` Python
text = "  Ура!  "

print(len(text))              # Результат: 8 (считает с пробелами)
print(text.strip())           # Результат: "Ура!" (убрал пробелы по бокам)
print("python".upper())       # Результат: PYTHON
print("Кот".replace("К", "М")) # Результат: Мот
````

```` Python
# Разрезаем текст по запятым
words = "Яблоко,Киви,Банан".split(",")
print(words)  # Результат: ['Яблоко', 'Киви', 'Банан']

# Склеиваем список обратно, вставив пробел между словами
text = " ".join(words)
print(text)   # Результат: "Яблоко Киви Банан"
````
