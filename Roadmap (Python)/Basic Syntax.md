### Python - синтаксис

Синтаксис Python определяет набор правил, которые используются для создания программ на Python. Синтаксис языка программирования Python во многом схож с синтаксисом языков программирования Perl, C и Java. Однако между ними есть и существенные различия.
### Идентификаторы Python

Идентификатор Python - это имя, используемое для обозначения переменной, функции, класса, модуля или другого объекта. Идентификатор начинается с буквы от A до Z, от a до z или с символа подчеркивания, за которым следует ноль или более букв, символов подчеркивания и цифр (от 0 до 9). В идентификаторах Python не допускаются такие знаки препинания, как &, $ и %.
Python - это язык программирования, чувствительный к регистру. Таким образом, Manpower и manpower - это два разных идентификатора в Python.
### Зарезервированные слова Python

Зарезервированные слова, которые нельзя использовать в качестве констант, переменных или других идентификаторов. Все ключевые слова Python состоят только из строчных букв.

### Отступы и блоки кода

В отличие от многих других языков, Python не использует фигурные скобки для обозначения блоков кода (тела функций, циклов, условий). Вместо этого применяется жестко контролируемая система отступов в начале строки. Все инструкции внутри одного блока должны иметь одинаковый размер отступа (одинаковое количество пробелов). Составные конструкции, такие как `if`, `while`, `def` или `class`, состоят из заголовочной строки, которая обязательно заканчивается двоеточием `:`, и последующего блока кода, выделенного отступом. Отступ составляет 4 пробела или tab. 
### Базовые типы данных и Операторы

Тип данных - это характеристика, которая определяет, какого рода значения могут храниться в переменной, сколько памяти они занимают и какие операции с ними можно выполнять.
Оператор - это специальный символ или ключевое слово, выполняющее определенное действие (вычисление, сравнение, присваивание) над одним или несколькими значениями (операндами)

Основные типы данных:

- int (integer / целое число): Любые целые числа без дробной части (положительные, отрицательные или ноль). _Пример: `count = 10`, `temperature = -5`._
- float (floating-point / число с плавающей точкой): Вещественные (дробные) числа. Разделителем всегда служит точка. _Пример: `pi = 3.14`, `price = 99.90`._
- str (string / строка): Набор символов, заключенный в одинарные или двойные кавычки. Используется для работы с текстом. _Пример: `name = "Python"`, `char = 'A'`._
- bool (boolean / логический тип): Логический тип данных, который может принимать всего два значения: `True` (истина) или `False` (ложь). Рождается в результате операций сравнения.
- list (список): Изменяемая упорядоченная коллекция элементов (аналог динамического массива). Может содержать данные разных типов. Заключается в квадратные скобки. _Пример: `users = ["Alex", "Ivan", "Elena"]`._
- dict (dictionary / словарь): Изменяемая структура данных, хранящая информацию в формате пар «ключ: значение». Ключ уникален и используется для быстрого доступа к значению. Фигурные скобки. _Пример: `car = {"brand": "Ford", "year": 2024}`_

### Условные операторы, циклы и функции

**Условные операторы** — это команды, которые позволяют программе принимать решения и выполнять разный код в зависимости от ситуации.
**Циклы** — это команды, которые заставляют один и тот же кусок кода повторяться несколько раз.
**Функция** — это отдельный именованный блок кода, который решает конкретную задачу. Его можно один раз написать, а потом вызывать в программе сколько угодно раз.

Условные операторы (`if - elif - else`):

Они проверяют условия на истинность. Если условие верно (`True`), код под ним срабатывает

- `if` (если) — основная и самая первая проверка.
- `elif` (сокращение от _else if_ / а если) — дополнительные проверки, если первый `if` не подошел. Их может быть сколько угодно.
- `else` (иначе) — финальный блок. Срабатывает сам, если ни одна проверка выше не сработала.

Циклы (`while` и `for`):

Нужны, чтобы не писать одну и ту же строчку кода по сто раз вручную

- **Цикл `while` (пока):** Крутится до тех пор, пока условие остается верным. _Пример: пока здоровье босса больше нуля — продолжать атаковать._
- **Цикл `for` (для):** Идеально подходит, чтобы перебрать элементы в списке (массиве) или повторить действие строго заданное число раз (обычно используется в связке с `range()`).

Функции (`def`):

Главный инструмент для того, чтобы не дублировать код в разных частях программы

- **Объявление:** Пишется через `def`, далее идет имя функции и в скобках - параметры (то, что мы передаем ей на вход).
- **Возврат значения (`return`):** Команда, которая отдает результат работы функции обратно в то место программы, откуда её вызвали.
