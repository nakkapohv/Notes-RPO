### Введение 

Циклы — это базовые конструкции языка, которые позволяют выполнять один и тот же блок кода многократно. В Python они избавляют разработчика от рутинного дублирования строк, помогая соблюдать главный принцип программирования. Вместо того чтобы писать сто одинаковых команд вручную, программист задает правило, и интерпретатор сам прокручивает код нужное количество раз. Каждое отдельное повторение кода внутри цикла называется **итерацией**
### While

Цикл `while` (в переводе «пока») выполняет код до тех пор, пока проверяемое логическое условие остается истинным (`True`). Перед каждым новым кругом Python проверяет это условие: если оно верно, код внутри цикла (тело цикла) выполняется, а если стало ложным (`False`), цикл тут же завершается, и программа идет дальше. Этот цикл идеален для ситуаций, когда заранее неизвестно точное количество повторений — например, когда нужно опрашивать датчик, пока он не зафиксирует движение, или просить пользователя ввести корректный пароль. Главное при работе с `while` — внутри тела цикла обязательно менять переменную, которая проверяется в условии, иначе цикл станет бесконечным и программа зависнет.
### For 

В отличие от многих других языков, где `for` работает как обычный числовой счетчик, в Python этот цикл устроен гораздо умнее. Он работает по принципу «для каждого элемента в коллекции» и последовательно перебирает содержимое любого объекта: буквы в строке, элементы в списке или ключи в словаре. На каждом шаге цикл сам берет очередной элемент и записывает его в рабочую переменную. Если же нужно просто повторить действие конкретное количество раз, `for` используют в связке с функцией `range()`, которая генерирует нужный диапазон чисел на лету (например, `range(5)` выдаст числа от 0 до 4). Цикл `for` считается в Python более безопасным и популярным, так как он гарантированно завершается сам, когда элементы в объекте заканчиваются.
### Инструменты

Иногда стандартный ход цикла нужно изменить прямо во время его работы, не дожидаясь финала. Для этого в Python есть два оператора, которые обычно используют внутри условного выражения `if`. Оператор `break` мгновенно и полностью останавливает работу всего цикла, выбрасывая программу из него (это полезно, например, когда при переборе списка мы нашли нужный элемент и дальше искать нет смысла). Оператор `continue` действует мягче: он не останавливает цикл, а лишь досрочно завершает текущую итерацию — программа пропускает оставшийся внизу код текущего круга и сразу перескакивает на начало следующего.

```` Python
hp = 3  # Количество жизней в игре

while hp > 0:
    print(f"Играем! Осталось жизней: {hp}")
    hp -= 1  # Уменьшаем на 1 каждый круг, чтобы избежать бесконечного цикла

print("Игра окончена.")
````

```` Python
# Перебор списка с любимой едой
favorite_foods = ["Пицца", "Бургер", "Суши"]

for food in favorite_foods:
    print(f"Я обожаю есть: {food}")

# Если нужно просто повторить действие несколько раз:
for step in range(3):
    print("Просто повтори это 3 раза")
````

```` Python
games = ["Тетрис", "Марио", "Doom", "Portal"]

for game in games:
    if game == "Doom":
        print("Нашли Doom! Дальше не ищем.")
        break  # Полностью вырубает цикл
        
    print(f"Проверяем игру: {game}")
````
