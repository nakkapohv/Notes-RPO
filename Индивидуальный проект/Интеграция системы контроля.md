### GitHub

Позволяет:

- Создавать репозитории — места для хранения проекта, в которых содержатся файлы с кодом. Репозитории могут быть публичными и приватными

- Отслеживать изменения в репозитории с помощью Git. Разработчик выполняет разные команды (например, commit, push), а все изменения синхронизируются с центральным репозиторием

- Использовать GitHub как хранилище проектов для портфолио — можно дать на них ссылку

Git помогает ориентироваться в коде и отслеживать изменения. С его помощью можно вернуть файлы в исходное состояние и видеть изменения, внесённые в определённый период.

### Канбан - доски

На Python можно использовать библиотеки и фреймворки для создания канбан-досок — инструментов для управления задачами в стиле канбан. Такие решения могут быть в виде консольных инструментов или веб-приложений. Канбан-доска помогает визуализировать движение задач — от планирования до завершения. Некоторые особенности:

- Доска разделена на столбцы, соответствующие этапам выполнения: «Планирование» (To Do), «В работе» (In Progress), «Готово» (Done)
- Карточки перемещаются по колонкам по мере выполнения, что позволяет контролировать процесс и видеть состояние проекта в реальном времени
